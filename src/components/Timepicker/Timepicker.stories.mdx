# Timepicker

Some explanation about our Timepicker component

## Preview of component

<Preview>
	<Story name='Base'>
		{() => {
			const [value, setValue] = useState('10:30');
			return (
				<Timepicker
					id="time"
					value={value}
					onChange={(newValue) => {
						action()(newValue);
						setValue(newValue);
					}}
				/>
			);
		}}
	</Story>
</Preview>
